# YJing（忆境空间）隐私政策

生效日期：2026-04-12

YJing（忆境空间）是一款 iOS 本地私密媒体管理应用，用于导入、整理、浏览和保护您设备上的照片、视频及其他文件。我们重视您的隐私，并尽量采用“本地优先”的设计。

本隐私政策说明我们会处理哪些信息、这些信息如何被使用，以及您可以如何联系我们。

## 1. 我们处理的信息

### 1.1 您在 App 中导入或管理的内容

当您使用 YJing 时，您可能会在 App 中导入、浏览、整理或传输以下内容：

- 照片、视频、PDF 及其他文件
- 文件夹结构、收藏信息、隐藏文件标记
- 您主动发起的移动、复制、删除、重命名等文件操作结果

这些内容主要保存在您设备上的 App 沙盒目录中，用于实现文件管理、浏览、搜索、缩略图展示、收藏、隐藏文件和局域网传输等功能。

### 1.2 安全与隐私相关信息

如果您启用了安全功能，App 还会处理以下信息：

- PIN 码或手势密码的哈希值及随机 salt
- 锁定状态、自动锁定时长、失败冷却状态
- Face ID / 生物识别解锁结果

其中，PIN/手势并不会以明文形式保存，而是以哈希后的形式存储在 Apple Keychain 中。Face ID 验证由 Apple 提供的系统框架处理，我们不会接收或保存您的面容数据。

### 1.3 本地偏好设置

为了保存您的使用习惯，App 会在本地存储部分偏好信息，例如：

- 主题、语言、排序和视图偏好
- 隐私偏好（如是否显示隐藏文件、是否在回到前台时自动隐藏）
- 摇一摇手势偏好
- 收藏、隐藏文件元数据以及部分界面状态

这些信息主要保存在设备本地的 UserDefaults、JSON 文件或系统 Keychain 中。

### 1.4 局域网 Wi-Fi 文件传输信息

如果您主动开启“WiFi 文件传输”功能，App 会在您的设备上临时启动一个仅供局域网访问的本地服务，以便您在同一网络中的浏览器里访问和管理文件。

在该功能下，App 可能会处理：

- 您设备当前的局域网访问地址和端口
- 一次性登录 PIN
- 用于维持当前局域网会话的本地 Cookie
- 您在浏览器端发起的上传、下载、浏览和文件管理请求

该功能默认仅在您手动开启时可用，且服务运行在您的设备本机，不通过我们运营的远程服务器中转。

### 1.5 订阅与应用内购买信息

如果您在 App 中购买订阅、恢复购买或查看会员状态，Apple 以及我们使用的第三方订阅服务 RevenueCat 可能会处理与购买相关的信息，例如：

- App Store 收据和交易状态
- 商品标识符、订阅状态、权益状态
- 用于识别购买状态的匿名 App User ID 或设备级标识
- 恢复购买所需的最少必要信息

这些信息用于：

- 验证您是否拥有有效订阅或对应权益
- 支持恢复购买
- 在您的设备上正确展示会员状态
- 处理基础的订阅运营和客服排查

我们不会因为订阅功能而主动要求您创建独立账号，除非未来某项功能明确要求。

## 2. 我们如何使用这些信息

我们处理上述信息主要是为了：

- 提供文件导入、浏览、整理、搜索、播放和传输功能
- 提供隐藏文件、锁屏保护、PIN/手势/Face ID 解锁等隐私与安全功能
- 保存您的个性化设置和使用偏好
- 提供订阅购买、权益验证和恢复购买能力
- 进行必要的故障排查、客服支持与安全防护

## 3. 我们如何共享信息

我们不会将您的照片、视频或其他文件内容出售给第三方。

在以下情形下，相关信息可能由第三方处理：

- **Apple**：当您使用系统能力（例如 Face ID、App Store 购买、iCloud/系统备份、照片或文件选择器）时，Apple 可能按照其自身隐私政策处理相关信息。
- **RevenueCat**：当您使用订阅、购买或恢复购买功能时，RevenueCat 可能代表我们处理与订阅状态验证相关的有限信息。

除上述情形外，您的文件内容通常不会被发送到我们自建的远程服务器；YJing 的核心媒体管理能力以本地处理为主。

## 4. 数据存储与保留

- 您导入到 App 中的文件和相关元数据通常保存在您的设备本地。
- 安全凭据的哈希信息保存在 Apple Keychain 中。
- 偏好设置和部分功能元数据保存在设备本地。
- 订阅相关信息可能由 Apple 和 RevenueCat 根据各自的数据保留规则处理。

只要您继续使用 App，相关本地数据通常会保留在设备上；当您删除相关内容、关闭相关功能或卸载 App 后，部分本地数据会被删除，但系统备份、Apple 或 RevenueCat 侧留存的数据可能仍受其各自规则约束。

## 5. 您的选择与控制

您可以通过以下方式管理自己的信息：

- 在 App 内删除文件、收藏或隐藏标记
- 关闭或调整 PIN、手势、Face ID 和自动锁定等安全功能
- 关闭 WiFi 文件传输功能
- 通过 App Store 管理、取消或恢复订阅
- 直接卸载 App，以移除大部分保存在设备本地的数据
