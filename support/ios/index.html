适用平台

本支持页同时适用于 YJing 的 iPhone / iPad 版和 Mac 版。两端通过 App Store 通用购买共用同一订阅权益，多数功能通用，少数功能仅在特定平台提供（例如 WiFi 文件传输、摇一摇手势仅 iOS 版；文件夹级独立 PIN、Touch ID 解锁仅 Mac 版）。

常见问题

Q：忘记 PIN / 手势密码怎么办？
出于隐私保护，密码仅本地存储且不可找回。在 Mac 版中，您可以使用 Touch ID 或系统登录密码重置 App PIN 和文件夹 PIN；在 iOS 版中，若确实忘记，请卸载并重新安装 App（注意：这会清空应用内数据）。

Q：WiFi 网页管理无法连接？（仅 iOS 版）
请确保 iPhone / iPad 和电脑处于同一 WiFi 网络下，并在 App 内打开"WiFi 管理"后访问屏幕上显示的地址。Mac 版当前不提供 WiFi 网页管理功能，您可以直接通过 Finder 拖拽、"打开..."或菜单导入文件。

Q：如何在 Mac 版中导入文件？
在 Mac 版中，您可以从 Finder 直接拖拽文件 / 文件夹到 YJing 窗口，或使用工具栏的"导入"按钮选择本地文件，也可以通过"照片选择器"从系统照片图库导入。

Q：从系统相册导入后原文件会被删除吗？
不会。导入操作仅复制到 YJing 私有空间，系统相册或原始文件夹中的原文件保持不变。

Q：Pro 订阅如何恢复？能否在 iPhone 和 Mac 之间共享？
可以。YJing 的 iOS 版和 Mac 版通过 App Store 通用购买共用订阅权益，使用同一 Apple ID 登录 App Store 即可。若权益未正确显示，请在"设置 → 恢复购买"中点击恢复；若仍有问题请发邮件联系我们。

Q：Mac 版的文件夹级 PIN 是怎么回事？
Mac 版支持为单独的文件夹设置独立的锁定 PIN，与整个 App 的 PIN 相互独立。您可以仅锁定部分敏感文件夹，进入这些文件夹时会要求单独验证，适合多人共用同一台 Mac 的场景。

隐私与数据

YJing 所有数据均存储在您的设备本地（iPhone / iPad / Mac 沙盒目录），我们不会收集、上传或分享您的照片、视频及文件。更多细节请参阅隐私政策。
